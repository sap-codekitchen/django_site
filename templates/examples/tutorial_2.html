Necessary Preparation

Please make sure you have everything installed before the workshop.

  1. Install a text editor.

  2. Open your command line (called "Terminal" on Mac and Linux) and get a basic familarity with your command line, using this tutorial. Get comfortable. This is your new tool.

    If you have Windows, download Git for Windows . When you get to the "Adjusting your Path Environment Option", leave the default option selected: "Use Git Bash only". Keep everything else as the default options. Once installed, open the Git Bash shell, and this will work similar to a Mac or Linux command line.
