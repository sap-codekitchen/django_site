What is an API?

Getting geolocated photos from Instagram

Necessary Preparation

  1. Install a text editor.

  2. Skim this tutorial on basic command line commands. Then open your command line (called "Terminal" on Mac and Linux)and get a basic familarity with your command line. Get comfortable. This is your new tool.

  3. Download the required files for this workshop and put them in a folder.

    This folder should contain the following files:

    folder/
    # a Python file
    getstuff.py

    # a bunch of json files
    beyonce_feed.json
    bylocation_cambbridge.json
    hashtag_harvard.json
    locationsearch_cam.json
    namesearch_beyonce.json

    # a folder with images
    beyonce_feed.json_images/
        0.jpg
        1.jpg
        2.jpg
        ...
        9.jpg
