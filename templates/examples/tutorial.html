Necessary Preparation

  1. Make sure you have a text editor installed. A text editor is a type of software used for editing text. TextEdit and Notepad are examples of very simple text editors that come pre-installed on most computers. Install something better. Here are some recommended text editors for beginners:

    • Sublime Text. SublimeText is an excellent text editor for beginners. It is free to try, and it will never force you to pay the $70 price. Windows, Mac, or Linux.
    • Atom. Atom is an extendable text editor created by GitHub. It is free, open source, and available for Mac and Windows.
    • Notepad++. Notepad++ is a general use text editor for Windows only.

  2. Skim this tutorial on basic command line commands. Then open your command line (called "Terminal" on Mac and Linux)and get a basic familarity with your command line. Get comfortable. This is your new tool.

  3. Download the required files for this workshop and put them in a folder.

    This folder should contain the following files:
