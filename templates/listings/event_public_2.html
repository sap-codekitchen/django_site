Secret Coding Essentials for Beginners

Basic skills that most tutorials leave out

Ben Golder will lead a workshop on basic skills for absolute beginners.

A lot of coding tutorials make unspoken assumptions about what a reader already knows. This workshop aims at demystifying the hardest parts of getting started so that you are better equipped to apply what you learn in tutorials. We will cover the basics of the command line, text editors, servers, data types, and package managers. This workshop makes other workshops and tutorials easier.

As usual, there will be snacks.
