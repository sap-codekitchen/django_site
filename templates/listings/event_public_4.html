Make Your First Python Script

taught by Moa Carlsson

Are you completely new to programming? Not to worry! This session will offer beginners an introduction to the Python programming language. We will cover basic data types, syntax, commands and operations—all with the aim to enable you to write your own simple python programs. In the second half of the session we will explore data structures with particular emphasis on the "list" data type. As you will see: understanding how to work with lists is essential for the handling of almost any data sets (big or small!) in most Python programs. No prior knowledge is necessary—looking forward to seeing you on Sunday!

This workshop expects you to review minimal command line skills and have a decent text editor installed.

As usual, there will be snacks.
