Python for 3D data visualization

taught by Carlos Sandoval

This workshop will show you how you can use Python to work with geometry and data. Carlos Sandoval, a SMArchS Design + Computation student, will lead you through a process of building 3d visualizations based on GIS data, using Python along with Grasshopper and Rhino, and the Decodes geometry library.

This workshop expects you to have a very basic familiarity with Python, or that you've at least skimmed the basics in this pdf . You should have Rhino for Windows installed (use the trial version if you don't already have it installed, the Mac OS version will not work for this tutorial), as well as the free Grasshopper plugin . You can download the workshop materials here, and the presentation slides can be viewed here.
