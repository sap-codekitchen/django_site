Make Your First Web Site

HTML, CSS, and how websites work

Building on our previous workshop on secret coding essentials, this workshop will walk you through the process of building a website using HTML and CSS, and displaying it to the world using your MIT user account. We will also briefly discuss webhosting, Content Management Systems (like Wordpress and Drupal), web frameworks (like Django and Ruby on Rails), and how your web browser gets a webpage from a server. This workshop requires minimal command line skills and a decent text editor.

As usual, there will be snacks.
