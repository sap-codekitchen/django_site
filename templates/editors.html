Text Editors: Essential for coding

Code is special kind of text. Coding is writing. To write code, you need something made for editing many different types of text files.

A text editor is a type of software used for editing text. TextEdit and Notepad are examples of very simple text editors that come pre-installed on most computers. Install something better. Here are some recommended text editors for beginners:

  • Sublime Text. SublimeText is an excellent text editor for beginners. It is free to try, and it will never force you to pay the $70 price. Windows, Mac, or Linux.
  • Atom. Atom is an extendable text editor created by GitHub. It is free, open source, and available for Mac and Windows.
  • Notepad++. Notepad++ is a general use text editor for Windows only.
  • Brackets is made for web programming and is easy to learn.
