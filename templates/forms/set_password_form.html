Note that this website is not using HTTPS encryption. This means that if someone wanted to snoop on the website with simple hacking tools, they could see the text that you enter on this form, including the password you enter. If someone hacks your CodeKitchen account through this there's not much at stake, but if you use the same password that you use on other sites, they could use the password to hack other accounts. To be safe, please use a password that you don't use elsewhere.

I wish we could setup HTTPS encryption, but it is difficult to do so on MIT's servers.
